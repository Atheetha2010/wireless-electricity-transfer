# Wireless Electricity Transfer ⚡

Short range wireless electricity transfer system using inductive coupling 
between two copper coils to power an LED across a distance of 6cm — 
no physical connection required.

## ⚙️ How It Works
A transmitter coil carries an alternating current, generating an oscillating 
magnetic field. A receiver coil placed 6cm away picks up this field through 
electromagnetic induction, generating enough current to power an LED — 
demonstrating wireless power transfer through resonant inductive coupling.

## 🔧 Components
| Component | Purpose |
|---|---|
| Copper Coil (transmitter) | Generates oscillating magnetic field |
| Copper Coil (receiver) | Picks up magnetic field and converts to current |
| LED | Powered wirelessly by receiver coil |
